# 歪fChat扫码端

## 一、项目介绍

该项目使用uniapp开发，主要用于扫码登陆桌面端软件

## 二、项目目录

- source: 项目源代码
- mp-weixin: 微信小程序

### 1. `source`: 推荐使用`HBuilderX`运行项目

1. 安装依赖

```bash
npm install
```

2. 需要在`scan-qrcode-app/source/common/url.js`文件中，将`BASE_URL`换成局域网ip

3. 运行到`微信开发者工具`

### 2. `mp-weixin`: 推荐使用`微信开发者工具`运行项目

填入自己的`appId`即可直接运行
